## Featured Work / Proyectos destacados

### Computer Vision & Applied AI

- [XAI-Crack-Detection](https://github.com/garay54/XAI-Crack-Detection) — Explainable AI utilities for YOLO crack detection with EigenCAM and LIME.  
  Herramientas de IA explicable para interpretar detecciones de grietas con EigenCAM y LIME.

- [CracksFIS](https://github.com/garay54/CracksFIS) — Fuzzy logic system for evaluating cracks using width, length, completeness, and density.  
  Sistema de logica difusa para evaluar grietas a partir de anchura, longitud, completitud y densidad.

- [SemanticNAS_Training](https://github.com/garay54/SemanticNAS_Training) — From-scratch semantic segmentation training package with Optuna-based NAS.  
  Paquete de entrenamiento desde cero para segmentacion semantica con busqueda de arquitecturas mediante Optuna.

- [Train_YOLOv11](https://github.com/garay54/Train_YOLOv11) — Automated YOLOv11 instance segmentation training with hyperparameter optimization.  
  Entrenamiento automatizado de segmentacion de instancias con YOLOv11 y optimizacion de hiperparametros.

- [CrackCuantificationFromImages](https://github.com/garay54/CrackCuantificationFromImages) — Crack detection and quantification pipeline from images and RealSense captures.  
  Pipeline para detectar y cuantificar grietas a partir de imagenes y capturas RealSense.

- [Grabador-Real-Sense-D435if](https://github.com/garay54/Grabador-Real-Sense-D435if) — Capture and extraction tools for Intel RealSense D435i/D435if workflows.  
  Herramientas de captura y extraccion para flujos de trabajo con camaras Intel RealSense D435i/D435if.
